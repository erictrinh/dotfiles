# Install

To bootstrap a new system, run

```shell
/bin/zsh <(curl -s https://raw.githubusercontent.com/erictrinh/dotfiles/master/install.sh)
```

You can also clone this repo and run `./install.sh`. Run `./osx.sh` for OSX defaults.
